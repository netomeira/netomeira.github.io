Primeiros passos com Bootstrap

Bootstrap CDN

Para começar a utilizar o Bootstrap em seu projeto podemos realizar o download da última versão pelo site oficial do framework, porém há outra forma de utilizá-lo sem precisar ter os arquivos no seu servidor de hospedagem. Para isso, seria necessário fazer uso do Bootstrap CDN que é uma forma de referência aos arquivos do projeto através do MaxCDN (Max Content Delivery Network), um provedor de conteúdo via rede muito usado. Veja no exemplo abaixo como realizar este procedimento em seu projeto.

Botões

Bootstrap inclui uma série de estilos de botões predefinidos, cada um com seu propósito semântico, com alguns extras para um maior controle.

Cards

Os cards são containers de conteúdo flexíveis e extensíveis. Eles incluem opções para headers e footers, uma grande variedade de conteúdo, cores de background contextuais, e poderosas opções de display.

List group
