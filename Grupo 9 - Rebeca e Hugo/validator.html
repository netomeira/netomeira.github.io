Dicas de Acessibilidade

Certifique-se sempre, de que o título da página seja descritivo. Utilize um verificador de sintaxe para validar o documento, é importante por razões de acessibilidade e renderização.

Quando as informações de estilo são removidas de um documento da Web ou os estilos são desativados, certifique-se de que o significado do conteúdo e das relações estruturais sejam preservados.
Ao projetar páginas da Web, considere que scripts e objetos programáticos apresentam possíveis problemas de acessibilidade para os usuários:

  1. alguns agentes do usuário, como o navegador somente texto Lynx, não suportam scripts;
  2. scripts podem causar problemas nos leitores de tela;
  3. os scripts às vezes são desabilitados em escolas, bibliotecas e locais de acesso público.



Linha 162

Todo conteúdo não textual deve conter uma alternativa em texto que possa ser utilizada como impressão ampla, braille, fala e etc.
Considere o uso da tag noscript para fornecer a alternativa em texto. O conteúdo desta tag será renderizado se os scripts estiverem desativados ou não forem suportados.

Linha 10

A tag "div" deve ser visto como um elemento de último recurso, para ser usado quando nenhum outro elemento é adequado.
Por exemplo, considere "article" para algo como um post de blog, "section" para um capítulo, "nav" para seções de navegação e "fieldset" para um grupo de controles de formulário.
A utilização de elementos mais apropriados, quando possível, pode melhorar a acessibilidade e a capacidade de manutenção dos autores.

Linha 18

O texto de um placeholder é geralmente ruim para acessibilidade, pois ele tende a desaparecer quando o usuário clicar no campo. A tag input não possui um atributo id.
Certifique-se de que todo o conteúdo não textual apresentado ao usuário tenha uma alternativa em texto que faça relação ao conteúdo.

Linha 12

Esta tag de título "h1" não contém texto detectável. Os títulos são importantes e devem conter o texto apropriado do cabeçalho.
As imagens podem exigir um atributo "longdesc" para fornecer um URI que contenha uma descrição longa que descreva a imagem.

Linha 27

A tag ul não é permitida como um filho da tag menu neste contexto

Linhas 39 e 52

A tag img possui um atributo alt vazio. Certifique-se de que imagens tenham texto alternativo.
